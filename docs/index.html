Git Interactive Rebase GUI Tool

A visual Git interactive rebase GUI built with PySide6. Reorder, squash, rephrase, drop, and split commits, inspect diffs, and manage Git history with a faster, cleaner interface than editing rebase instructions by hand.

Get Started Star on GitHub

✨ Git Rebase Features

🛠️

Visual Interactive Rebase

Drag and drop commits to reorder history. Squash, rephrase, drop, and manage interactive rebase actions with a clear visual editor.

🔍

Search, Filter, and Inspect

Search commits by SHA or message, inspect commit details, and navigate history quickly with keyboard shortcuts: / Esc F5.

🎨

Dark and Light Themes

Use adaptive dark and light themes inspired by modern IDEs, with persistent settings for a smoother Git workflow.

⚡

Diffs, Split, and Efficient History Editing

Inspect diffs, work with commit ranges, use headless execution, and avoid messy terminal editing during complex Git history rewriting.

🚀 Quick Start

# Install via pip
pip install git+https://github.com/shyjun/git-interactive-rebase-gui-tool.git

# Run the app
git_interactive_rebase

Or clone the repository for the latest development version:

# Clone and run
git clone https://github.com/shyjun/git-interactive-rebase-gui-tool.git
cd git-interactive-rebase-gui-tool
pip install PySide6
python git_interactive_rebase.py
